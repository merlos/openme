# openme — iOS App

> iOS/iPadOS client · **Status: Planned**

## Planned Stack

- **Language:** Swift
- **UI:** SwiftUI
- **Min iOS:** 16
- **Distribution:** App Store

## Planned Features

- Profile manager
- Home Screen widget and shortcut for one-tap knock
- QR code scanner for server config import
- On-device Ed25519 key generation via Secure Enclave
- Face ID / Touch ID authentication before knock
- Shortcuts app integration
- iCloud Keychain sync for profiles across Apple devices (opt-in)
