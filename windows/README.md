# openme — Windows GUI

> Native Windows desktop client · **Status: Planned**

## Planned Stack

- **Language:** C# / .NET 8
- **UI:** WinUI 3 (Windows App SDK)
- **Packaging:** MSIX

## Planned Features

- Profile manager (add, edit, remove server profiles)
- System tray — knock without opening the main window
- One-click connect buttons per profile
- QR code scanner for importing configs from mobile
- Auto-knock on network change (wake from sleep, VPN connect)
- Bundled `openme.exe` core for packet sending
